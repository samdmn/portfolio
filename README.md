# Mon Portfolio

J'ai developpé ce portfolio lors de ma recherche de stage de fin de Licence Informatique. Je le vois comme une suite améliorée de CV, pour pouvoir fournir plus de détails à quiconque souhaite se renseigner sur moi.

## Description

J'ai scindé le portfolio en plusieurs parties claires :
- Projets : cette partie répertorie les projets que j'ai pu faire en cours ou par moi-même, avec une courte description de chaque projet ainsi que les langages utilisés
- Expérience Académique : un résumé de ma formation académique
- Expérience Professionelle : un résumé de mon expérience professionelle
- Contact : une page listant les moyens de me contacter (Mail, LinkedIn, Github)
- CV : pour pouvoir visualiser le PDF de mon CV si besoin 

## Technologies utilisées

- **Language :** HTML, CSS  
- **Documentation (ce README)** : Markdown
